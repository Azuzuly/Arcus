# Arcus

A modern AI chat interface with dark glass-morphism design.

## Stack
Plain HTML + CSS + JS â€” no build step needed.

## Features
- Home / Studio / Agent tabs
- Collapsible sidebar with chat list
- Model selector dropdown with search
- Chat settings panel (Temperature, Max Tokens, Top P, Top K, Frequency & Presence Penalty)
- Settings page (Account, Appearance, System Prompt, Memory, Usage, About)
- Upgrade / Pro plans modal
- Pro upsell popup
- Rename & Delete chat modals
- Feedback modal
- File upload popup
- Floating trigger menu (Settings, Upgrade, Feedback)
- Upgrade banner
- New chat creation

## Deploy
Drop into [Vercel](https://vercel.com) or open `index.html` directly in your browser.
